# openEffect

Web 上で動く After Effects 風モーショングラフィックスツールです。
UI/UX・機能・用語を可能な限り Adobe After Effects(日本語版)に近づけて再現しています。
ビルド不要・依存ゼロの純粋な HTML/CSS/JavaScript(ES Modules)で書かれており、ブラウザだけで動作します。

## デモ

GitHub Pages: **https://dikmri.github.io/openEffect/**

## 主な機能

### プロジェクト / コンポジション
- 新規プロジェクト / 保存(JSON ダウンロード + localStorage 自動保存)/ 開く
- 新規コンポジション(プリセット: HDTV 1080 29.97, NTSC DV, Web, 正方形, 縦型 など)
- コンポジション設定(幅・高さ・フレームレート・デュレーション・背景色)
- 画像 / 動画フッテージの読み込み(ドラッグではなくメニューまたはプロジェクトパネルから)
- プロジェクトパネル(検索・種類アイコン・サイズ・デュレーション表示)

### タイムライン
- レイヤー(映像 / ソリッド / テキスト / シェイプ)の追加・複製(Ctrl+D)・分割(Ctrl+Shift+D)・削除
- レイヤーバーのドラッグ移動・両端トリム(動画はソース範囲を考慮)
- スイッチ: 表示(目)・ソロ・ロック・シャイ(マスター切替付き)
- ブレンドモード(通常/加算/スクリーン/乗算/オーバーレイ 他)
- トランスフォーム 5 プロパティ(アンカーポイント/位置/スケール/回転/不透明度)
- ストップウォッチによるキーフレーム有効化、キーフレームダイヤモンドの追加/選択/ドラッグ移動
- キーフレームナビゲーション(◄ ◆ ►)、イージーイーズ(F9)/リニア補間
- タイムルーラー・再生ヘッドのスクラブ・ワークエリア(B / N キー)
- 時間ズームスライダー、タイムコード表示(クリックで時間移動)
- 右クリックコンテキストメニュー(ラベルカラー・重ね順・リセット など)

### コンポジションビューア
- 選択ツールによる移動・8 ハンドルのスケーリング(Shift で縦横比固定)
- 回転ツール(W)、ハンドツール(H)、ズームツール(Z / ホイール)
- テキストツール(T)、長方形/楕円ツール(Q / Shift+Q)
- ズーム(12.5%〜800%・フィット)、解像度切替(フル/ハーフ/3分の1/4分の1)
- ダブルクリックでテキスト/ソリッド設定を編集

### エフェクト(Canvas 2D で実装)
- ぼかし・シャープ: ガウスぼかし、高速ボックスぼかし
- パースペクティブ: ドロップシャドウ
- スタイライズ: グロー、モザイク
- カラーコレクション: 色相/彩度、輝度・コントラスト、反転
- 生成: 塗りつぶし
- ディストーション: トランスフォーム
- エフェクト＆プリセット パネル(検索・ダブルクリックで適用)
- エフェクトコントロール パネル(パラメータ編集・fx 切替・並べ替え・リセット・削除)

### その他
- 取り消し/やり直し(Ctrl+Z / Ctrl+Y、40 ステップ)
- プレビュー再生(スペース)、ループ、ワークエリア範囲再生
- WebM 書き出し(ファイル > 書き出し、リアルタイム録画)
- パネル分割レイアウト(スプリッターでリサイズ可能)、ウィンドウメニューで表示切替

## ローカルでの実行

ES Modules を使っているため、静的サーバー経由で開いてください。

```bash
cd openEffect
npx serve .
# または
python -m http.server 8000
```

`http://localhost:8000` (または表示されたポート)にアクセス。

## 技術構成

- Vanilla JS (ES Modules) + Canvas 2D API
- 描画: コンポジションをオフスクリーンキャンバにレンダリング(キーフレーム評価 → レイヤー描画 → エフェクトチェーン → ブレンドモード合成)
- エフェクト: `ctx.filter` (blur / hue-rotate / saturate / brightness / contrast / invert) と合成オペレーションで実装
- 書き出し: `canvas.captureStream()` + `MediaRecorder`

```
  effectControls.js エフェクトコントロール/プリセットパネル
  dialogs.js        各種ダイアログ
  utils.js          ユーティリティ
```
